# Flask Tutorial

### Running the app

From Powershell:
`$env:FLASK_APP = "flaskr"`
`$env:FLASK_ENV = "development"`
`flask init-db` _Execute it just the first time_
`flask run`

From bash:
`export FLASK_APP=flaskr`
`$ export FLASK_ENV=development`
`flask init-db` _Execute it just the first time_
`$ flask run`

Open a browser and go to:
http://127.0.0.1:5000/
